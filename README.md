# Board
코딩학원 게시판 웹 교재 만들기

# Introduce
코딩학원 근무 중 웹을 접해본 적 없거나 흥미가 있던 학원생들을 위하여 온라인 교재를 작성한 경험이 있습니다. 입문하기에 쉬운 게시판 만들기로 내용을 작성하였으며, SpringBoot과 React를 사용하였습니다.
